Ю́лиус Вильге́льм Ри́хард Дедеки́нд (нем. Julius Wilhelm Richard Dedekind) — немецкий математик, известный работами по общей алгебре и основаниям вещественных чисел.

В 1871 году Дедекинд знакомится с Георгом Кантором. Знакомство перешло в долголетнюю дружбу и сотрудничество; Дедекинд стал одним из первых сторонников канторовской теории множеств, и многие его работы стали наглядным примером применения новых методов. Новаторским стало и широкое применение Дедекиндом аксиоматического подхода к описанию новых (абстрактных) математических понятий. В 1888 году Дедекинд предложил первый вариант системы аксиом для системы натуральных чисел.
