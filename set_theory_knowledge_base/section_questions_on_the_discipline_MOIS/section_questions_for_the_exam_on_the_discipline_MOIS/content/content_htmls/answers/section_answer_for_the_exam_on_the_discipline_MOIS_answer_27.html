Проекция отношения R по атрибутам A1, A2, ... An - множество знаков тех кортежей, которые строятся из кортежей отношения R путём удаления всех вхождений элементов, которые имеют атрибуты, не совпадающие с указанными атрибутами A1, A2, ... An, а также путём последующего устранения кратности кортежей (из семейства кратных кортежей в указанном множестве кортежей должен остаться 1 представитель).
Множество L называется проекцией отношения R по атрибуту A, если:
  • каждый элемент с атрибутом A кортежа, принадлежащего отношению R, является элементом множества L;
  • каждый элемент множества L является элементом с атрибутом A кортежа, принадлежащего отношению R.
Например, есть отношение: φ = <Ф, M>, M = {a, b, c, d}, Ф = {<a, c>, <d, b>, <b, a>,<c, b>}.
Тогда проекцией отношения по атрибуту 2 будет L = пр2 Ф = {a, b}.
Композиция отношения φ1 ⊆ A × B и φ2 ⊆ B × C называется множество:
  • φ1 · φ2 = { <x,y> | x ∈ A, y ∈ C ∧ ( ∃ z ∈ B <x,z> ∈ φ1, <z,y> ∈ φ2)}
Например, пусть есть два отношения: φ1 = <Ф1, M>, φ2 = <Ф2, M>, M = {1, 2, 3}, Ф1= {<1, 1>, <1, 2>, <1, 3>,<3, 3>}, Ф2 = {<1, 1>,<2, 3>, <3, 1>,<3, 2>}.
Тогда композиция графиков этих отношений равна Ф3 = Ф1Ф2 = {<1, 1>, <1, 3>,<1, 2>, <3, 2>, <3, 1>}.
Пусть отношения R1 и R2 не имеют одинаковых атрибутов, а символ Θ обозначает один из логических операторов. Тогда соединением отношения R1 по атрибуту X с отношением R2 по атрибуту Y является отношение, совпадающее с декартовым произведением отношений R1 и R2, для кортежей которых выражение условия X Θ Y - истина.
Например, пусть есть два отношения: φ1 = <Ф1, M>, φ2 = <Ф2, M>, M = {a, b, c}, Ф1= {<b, a>, <a, b>, <a, c>}, Ф2 = {<c, a>,<b, c>, <a, a>,<c, b>}.
Тогда соединение графиков этих отношений по атрибутам φ1_1 и φ2_1 при Θ = "=" равнo Ф3 = {<<b,a>, <b,c>>, <<a,b>, <a,a>>, <<a,c>, <a,a>>}.
