Алгебра - множество, на котором заданы только операции.
Полугруппа - это алгебра (A; (*)), где операция «*» удовлетворяет свойству ассоциативности: т.е.
∀a ∀b ∀c (({a, b, c} ⊆ A) →: (a * (b * c) = (a * b) * c)).
Группа - это алгебра (A; (*, -1, 1)), где операция «*» удовлетворяет свойству ассоциативности, «1» – единичный элемент относительно операции «*», а-1 – унарная операция, дающая обратный элемент относительно операции «*», т.е. (∀a (a ∈ A → (a * a-1 = a-1 * a = 1).
Кольцо - это алгебра с двумя операциями (M, (+, *)), где (M, (+)) группа, причем m1 * m2 = m2 * m1, и для любых m1, m2, m3 принадлежащих M: (m1 + m2) * m3 = (m1 * m3) + (m2 * m3).
Поле — алгебраическая система, для элементов которой определены операции сложения, вычитания, умножения и деления, и для которой выполняется следующее:

  • Коммутативность сложения: ∀a ∀b (a ∈ F ∧ b ∈ F → a + b = b + a)
  • Ассоциативность сложения: ∀a ∀b ∀c (((a ∈ F ∧ b ∈ F) ∧ c ∈ F) → (a + b) + c = a + (b + c)
  • Существование нулевого элемента: ∃0 ∈ F : ∀a ∈ F (a + 0 = a)
  • Существование противоположного элемента: ∀a ∈ F ∃(-a) ∈ F: a + (-a) = 0
  • Коммутативность умножения: ∀a ∀b (a ∈ F ∧ b ∈ F → a * b = b * a)
  • Ассоциативность умножения: ∀a ∀b ∀c (((a ∈ F ∧ b ∈ F) ∧ c ∈ F) → (a * b) * c = a * (b * c)
  • Существование единичного элемента: ∃e ∈ F : ∀a ∈ F (a * e = a)
  • Существование обратного элемента для ненулевых элементов: (∀a ∈ F: a ≠ 0) ∃a-1 ∈ F: a * a-1 = e
  • Дистрибутивность умножения относительно сложения: ∀a ∀b ∀c (((a ∈ F ∧ b ∈ F) ∧ c ∈ F) → ((a + b) * c = a * c + b * c))
