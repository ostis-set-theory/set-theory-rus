Модель - алгебраическая система, сигнатура которой содержит только отношения.
Графовая модель - алгебраическая система, сигнатура которой содержит только бинарные отношения.
Граф - алгебраическая система, сигнатура которой содержит только одно бинарное отношение - отношениями смежности. Связки этих отношений называют рёбрами графа. Первичные элементы графа называют вершинами.
Существует типология графовых структур (графов):
  • ориентированные графы,
  • неориентированные графы,
  • смешанные графы;
  • связные графы,
  • несвязные графы;
  • цикличные графы,
  • ацикличные графы
  • деревья
  • бинарные деревья,
  • тернарные деревья
  • простые графы;
  • пустые графы;
  • нуль-графы;
  • полные графы;
  • симметрическиe графы,
  • асимметрические графы;
  • планарные графы,
  • непланарные графы;
  • двудольные графы.
