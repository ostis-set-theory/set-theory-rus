Множество размещением (из n по m, n > m) называется множество всех различных упорядоченных наборов из m различных элементов из некоторого множества различных n элементов.
Количество размещений из n по m, обозначаемое Anm, равно убывающему факториалу:
  • Anm = n! / (n - m)!
Каждое размещение из n по m однозначно соответствует некоторому сочетанию из n по m и некоторой перестановкеэлементов этого сочетания; число сочетаний из n по m равно биномиальному коэффициенту (nm), в то время как перестановок на m элементах ровно m! штук.
При m = n количество размещений равно количеству перестановок порядка n:
  • Ann = n! / (n - n)! = n! / 1 = n!
Пример:
  • М = {1,2,3}
  • A23 = {<1,2>, <2,1>, <1,3>, <3,1>, <2,3>, <3,2>}
